# memo
memo is a command-line application to help memorize via a simple forgetting curve mnemonic. 

# Install
```
go install github.com/xuxinx/memo/cmd/memo@main 
```

# Usage
vim-like key bindings.  
press `ctrl+h` for each page's help info.
